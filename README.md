# Portfolio

A minimal, animated personal portfolio website built with modern web technologies. 
Designed with a clean, soft editorial aesthetic to perfectly showcase software engineering and full-stack projects.

## Tech Stack

- **Framework**: [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://motion.dev/)
- **Routing**: React Router v7

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   bun install 
   # or npm install
   ```
3. Run the development server:
   ```bash
   bun run dev
   # or npm run dev
   ```

## Customization

All static content (projects, copy, social links, and images) is neatly organized in `src/data.js`. Simply update the exports in that file to customize the portfolio for your own use without needing to edit the core React components!
